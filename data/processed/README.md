# Processed SurveilAMR outputs

These aggregated tables are derived from Vivli AMR Register datasets under Data Request **00013370**. They contain summary counts and resistance proportions only—not isolate-level raw data.

| File | Description |
|------|-------------|
| `dataset_summary.json` | ATLAS cohort overview and resistance trends |
| `surveilamr_resistance_by_year.csv` | Yearly resistance rates for priority pairs |
| `surveilamr_resistance_by_country.csv` | Country-level resistance rates |
| `supplementary_datasets_summary.json` | KEYSTONE / DREAM / GASAR summaries |
| `keystone_omadacycline_mic.csv` | Omadacycline MIC50/90 by pathogen |
| `keystone_omadacycline_by_year.csv` | Yearly omadacycline MIC50 trends |
| `dream_bdq_by_year.csv` | Bedaquiline MIC trends |
| `dream_bdq_by_continent.csv` | Bedaquiline MIC by continent |
| `gasar_phenotype_counts.csv` | GASAR phenotype buckets |
| `gasar_gene_counts.csv` | GASAR gene combinations |

Raw data: request access at https://amr.vivli.org
